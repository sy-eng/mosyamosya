つまり、全ての$p_k \geq 0$となるkに対して、以下が成り立つ。
\begin{equation}
{P_{(p_1, \cdots, p_n)({p'}_1, \cdots, {p'}_n)}}^{(r')}(C) = 
\sum_{m=1, p_m \geq 1}^{n}c_{km}{P_{(p_1, \cdots, (p_k -1), \cdots, p_n)({p'}_1, \cdots, ({p'}_m - 1), \cdots, {p'}_n)}}^{(r'-1)}(C)
\end{equation}
この式はCだけでなく、A,Bに対しても成立する。

\begin{equation}
c_{km} = \sum_{l}^{n}a_{kl}b_{lm}
\end{equation} 
\begin{equation}
\begin{split}
{P_{(p_1, \cdots, (p_k -1), \cdots, p_n)({p'}_1, \cdots, ({p'}_m - 1), \cdots, {p'}_n)}}^{(r'-1)}(C) = \\
\sum_{Pset_{all}} {P_{(p_1, \cdots, (p_k -1), \cdots, p_n)(Pset_{all})}}^{(r'-1)}(A){P_{(Pset_{all})({p'}_1, \cdots, ({p'}_m - 1), \cdots, {p'}_n)}}^{(r'-1)}(B)
\end{split}
\end{equation}
に注意して、 
\begin{equation}
\begin{split}
{P_{(p_1, \cdots, p_n)({p'}_1, \cdots, {p'}_n)}}^{(r')}(C) = \sum_{m=1, p_m \geq 1}^{n}c_{km}{P_{(p_1, \cdots, (p_k -1), \cdots, p_n)({p'}_1, \cdots, ({p'}_m - 1), \cdots, {p'}_n)}}^{(r'-1)}(C) = \\
\sum_{m=1, p_m \geq 1}^{n}(\sum_{l}^{n}a_{kl}b_{lm})(\sum_{{Pset_{all}}^{(r-1)}} {P_{(p_1, \cdots, (p_k -1), \cdots, p_n)({Pset_{all}}^{(r-1)})}}^{(r'-1)}(A){P_{({Pset_{all}}^{(r-1)})({p'}_1, \cdots, ({p'}_m - 1), \cdots, {p'}_n)}}^{(r'-1)}(B))
\end{split}
\end{equation}
一方、$P_{r'}(A)P_{r'}(B)$の$(p_1, \cdots, p_n)({p'}_1, \cdots, {p'}_n)$成分を考慮する。  
\begin{equation}
\begin{split}
\sum_{{Pset_{all}}^{(r)}}(\sum_{m^{'}=1, p_{m^{'}} \geq 1}^{n}a_{k^{'}m^{'}}{P_{(p_1, \cdots, (p_{k^{'}} -1), \cdots, p_n)({Pset_{all}}^{(r)}(p'_{m^{'}} - 1))}}^{(r'-1)}(A))\\
(\sum_{m^{''}=1, p_{m^{''}} \geq 1}^{n}b_{k^{''}m^{''}}{P_{({Pset_{all}}^{(r)}(p_{k^{''}} -1))({p'}_1, \cdots, ({p'}_{m^{''}} - 1), \cdots, {p'}_n)}}^{(r'-1)}(B))
\end{split}
\end{equation}
ここで、$\sum_{{Pset_{all}}^{(r)}}(\sum_{m^{'}=1, p_{m^{'}} \geq 1}^{n}a_{k^{'}m^{'}}{P_{(p_1, \cdots, (p_{k^{'}} -1), \cdots, p_n)({Pset_{all}}^{(r)}(p'_{m^{'}} - 1))}}^{(r'-1)}(A))$を
考慮すると、$\sum_{{Pset_{all}}^{(r-1)}}(\sum_{l}^{n}a_{k^{'}l}{P_{(p_1, \cdots, (p_{k^{'}} -1), \cdots, p_n)({Pset_{all}}^{(r-1)})}}^{(r'-1)}(A))$となる。
(一つ引いて、rで全体を走らせるのは、r-1で全体を走らせる場合に相当。)

また、$k^{'},k^{''}$は任意の数字にできるが、$k^{'} = k, k^{''} = m^{'} = l$として、列が$({p'}_1, \cdots, {p'}_n)$にて、定義されるので、
$m^{''} = m$となる。

よって、$P_{r'}(A)P_{r'}(B)$の$(p_1, \cdots, p_n)({p'}_1, \cdots, {p'}_n)$成分は、
\begin{equation}
\begin{split}
\sum_{{Pset_{all}}^{(r-1)}}\sum_{l}^{n}(a_{kl}{P_{(p_1, \cdots, (p_{k} -1), \cdots, p_n)({Pset_{all}}^{(r-1)})}}^{(r'-1)}(A))(\sum_{m=1, p_{m} \geq 1}^{n}b_{lm}{P_{({Pset_{all}}^{(r-1)})({p'}_1, \cdots, ({p'}_m - 1), \cdots, {p'}_n)}}^{(r'-1)}(B))
\end{split}
\end{equation}
ここで、a,bは${Pset_{all}}^(r-1)$に依存しないため、また、mはbにしか影響しないため、和の順番を入れ替えて、
\begin{equation}
\begin{split}
\sum_{m=1, p_{m} \geq 1}^{n}\sum_{l}^{n}(a_{kl}b_{lm})\sum_{{Pset_{all}}^{(r-1)}}({P_{(p_1, \cdots, (p_{k} -1), \cdots, p_n)({Pset_{all}}^{(r-1)})}}^{(r'-1)}(A){P_{({Pset_{all}}^{(r-1)})({p'}_1, \cdots, ({p'}_m - 1), \cdots, {p'}_n)}}^{(r'-1)}(B))
\end{split}
\end{equation}
となり、上記の${P_{(p_1, \cdots, p_n)({p'}_1, \cdots, {p'}_n)}}^{(r')}(C)$と一致する。
